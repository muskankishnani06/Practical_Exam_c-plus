# C++ OOP Practical Programs

A collection of **C++ Object-Oriented Programming (OOP)** practical programs demonstrating the core concepts of OOP such as **Classes, Objects, Encapsulation, Inheritance, Polymorphism, Virtual Functions, and Abstract Classes**.

## 📂 Project Structure

```
.
├── All_in_one.cpp     # Contains all practical programs
├── Que_1.cpp          # Book Management using Class & Objects
├── Que_2.cpp          # Person Details using Constructors & Encapsulation
├── Que_3.cpp          # Animal Inheritance Example
├── Que_4.cpp          # Runtime Polymorphism using Virtual Functions
├── Que_5.cpp          # Abstract Class and Pure Virtual Functions
└── README.md
```

---

## 📖 Programs Included

### ✅ Question 1 – Book Management System

**Concepts Used**
- Class & Object
- Array of Objects
- User Input
- Basic Data Storage

**Description**
- Stores details of multiple books.
- Accepts Title, Author, and Published Year.
- Displays all entered book information.

---

### ✅ Question 2 – Person Information System

**Concepts Used**
- Encapsulation
- Parameterized Constructor
- Getter & Setter Functions
- Dynamic Memory Allocation
- Array of Pointer Objects

**Description**
- Stores multiple person's details.
- Uses constructors for initialization.
- Retrieves data using getter methods.

---

### ✅ Question 3 – Animal Inheritance

**Concepts Used**
- Inheritance
- Base Class
- Derived Class
- Code Reusability

**Description**
- Creates Dog and Cat classes inherited from Animal.
- Demonstrates inherited functions.
- Displays the sound made by each animal.

---

### ✅ Question 4 – Runtime Polymorphism

**Concepts Used**
- Virtual Functions
- Method Overriding
- Runtime Polymorphism
- Dynamic Binding

**Description**
- Uses Animal as the base class.
- Dog and Cat override the `displayDetails()` function.
- Demonstrates polymorphism using base class pointers.

---

### ✅ Question 5 – Abstract Class

**Concepts Used**
- Abstract Class
- Pure Virtual Function
- Runtime Polymorphism

**Description**
- Shape is an abstract class.
- Circle and Rectangle implement the abstract methods.
- Calculates and displays the area of each shape.
